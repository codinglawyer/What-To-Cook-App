## What To Cook
##### (in the making)
- the application for managing your favorite recipes (recipes are saved in the Firebase realtime database)

### Getting Started

1. **Clone repo**

    `git clone https://github.com/codinglawyer/What-To-Cook-App`

2. **Install dependencies**

    `npm install`

3. **Start dev server**

  `npm start`

4. **Open your browser**

  `http://localhost:3000/`
