## What To Cook
##### (in the making)
- an application for managing your favorite recipes (recipes are saved in the Firebase realtime database)
- tests included

### Getting Started

1. **Clone repo**

    `git clone https://github.com/codinglawyer/What-To-Cook-App`

2. **Install dependencies**

    `npm install`

3. **Start dev server**

    `npm run dev`

4. **Open your browser**

    `http://localhost:8080/`
